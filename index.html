Тема урока ветки и комиты

  • Создание ветки - git branch namebranch
  • Слияние веток - Git merge навзание ветки
  • Переход и переход на Commit или ветку будет - git checkout
  • Удаление веток - git branch -d имя ветки
  • Создание ветки и сразу же переходить на ветку - git checkout -b имя ветки
  • Клонирование проекта - Git clone
  • Удаляет коммити - Git reset ссылка на коммит
  • Вывод истории коммитов сокращенно - Git log --oneline
  • Докачать файлы - Git pull
