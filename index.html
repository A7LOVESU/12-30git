Hellow, Git! We are super!

  • git init - это создание репозитория (хранилища ваших данных )
  • git config user.name and user.email - это знакомство с гит репозиторием
  • git add . || git add index.html - это добавление файлов в отслеживаемую зону
  • git status - это статус отслеживаемой зоны
  • git commit -m - это создание точки сохранения отслеживаемую зону
  • gitk - это просмотр репозитория
